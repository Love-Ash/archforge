# Accuracy record (public corpus)

Per-gate counts measured on the [public corpus](../corpus/), regenerated by
`python scripts/make_accuracy_table.py`. Every number is third-party
reproducible: the same decks and manifests ship in this repository, and CI
fails on any drift.

Honest scope: this corpus is a seed (18 decks from 3 generator families,
5 of them clean negatives). It proves the gates fire on constructed ground
truth and stay silent on clean decks; it is not yet a field-scale benchmark.
Growing it (Google Slides / Canva / LibreOffice exports, multi-master corporate
templates, contributed false positives) is the highest-value contribution this
project takes; see [corpus/README.md](../corpus/README.md).

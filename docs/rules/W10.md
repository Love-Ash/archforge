# W10: Hand-drawn diagram cloned across pages

| | |
|---|---|
| Severity | warning |
| Category | structure |
| Profiles | editorial, full |

## What it means

A hand-drawn diagram (decorative texture marks) is cloned nearly identically across pages.

## How to fix

Confirm by eye whether it is reuse or intent, then redesign.

## If you disagree

False-positive reports with a repro deck are the most valuable contribution this project takes: use the [FP template](https://github.com/Love-Ash/archforge/issues/new/choose). Calibration rationale lives in [docs/CALIBRATION.md](../CALIBRATION.md).

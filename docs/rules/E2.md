# E2: Dash-family character used as punctuation

| | |
|---|---|
| Severity | error |
| Category | style |
| Profiles | editorial, full |

## What it means

A dash-family character (em/en/figure dash, horizontal bar, math minus, fullwidth hyphen, box-drawing line) is used as sentence punctuation: the single most reliable tell of AI-generated copy. Numeric ranges (2020-2024, Q1-Q3, 5%-10%) and minus signs before digits pass by default; --e2-no-exemptions blocks everything.

## How to fix

Replace prose dashes with a colon, comma, parentheses, or a line break. Numeric ranges and minus signs pass by default.
